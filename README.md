# Coffee Delivery

Coffee Delivery is a web commerce for a coffe shop

## Requeriments
You need to have Node installed locally
## Usage

```bash
git clone https://github.com/tiago0214/coffee-delivery.git

cd coffee-delivery

npm install

npm run dev
```
